## Зависимости

- Ruby 2.3+ (MRI)
- Postgres 9.5 or 9.6

## Установка
1. Выполните `cp config/secrets.yml.examle config/secrets.yml`
2. `rake db:create`
3. Проверяем, что все работает запуском `rake`
